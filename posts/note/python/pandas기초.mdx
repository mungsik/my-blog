### DataFrame

        - **pd.Series( )**

            index, value

        - **pd.DataFrame( )**

            index, value, column

        - **loc( ) vs iloc( )**

            loc : location. index 이름으로 특정 행, 열을 선택

            iloc : inter location. 컴퓨터가 인식하는 인덱스 값을 선택

        - **axis**

            axis == 0 가로, axis == 1 세로

        - **데이터 만들기**

            1. 딕셔너리 안의 리스트 형태

                ```python
                left = pd.DataFrame({
                    "key" : ["K0", "K1", "K2", "K3],
                    "A" : ["A0", "A1", "A2", "A3"],
                    "B" : ["B0", "B1", "B2", "B3"]
                })
                ```


                - 결과. 열 값 기준으로 들어감

                ![](/note/python/dic-in-list.PNG)

            2. 리스트 안의 딕셔너리 형태

                ```python
                right = pd.DataFrame([
                    {"key":"K0", "C" : "C0", "D": "D0"}
                    {"key":"K1", "C" : "C1", "D": "D1"},
                ] )
                ```


                - 결과

                ![](/note/python/list-in-dic.PNG)
