## 풀이

```python
import sys
sys.stdin = open("input.txt", "r")
n = int(sys.stdin.readline())
for i in range(n):
    s = sys.stdin.readline()
    s = s.upper()
    for j in range(len(s) // 2):
        if s[j] != s[-1 - j]:
            print("#%d NO" % (i + 1))
            break
    else:
        print("#%d YES" % (i + 1))

```

혹은

```python

sys.stdin = open("input.txt", "r")
n = int(sys.stdin.readline())
for i in range(n):
    s = sys.stdin.readline()
    s = s.upper()
    if s == s[::-1]:
        print("#%d YES" % (i + 1))
    else:
        print("#%d NO" % (i + 1))
```

## 짚고 넘어갈 것

#### 1. slice

    - **[::-1]**

    리스트 전체에서 인덱스를 1씩 감소시키면서 요소를 가져오므로 리스트를 반대로 뒤집는다.
