- 리스트를 이용한 방법

```python
n = 4
m = 6

cnt = [0] * (n + m + 3)  # n+m 만큼을 리스트의 길이로 정함. 여기서 +3은 대략적으로 여유공간 준 것.
max = float("-inf")

for i in range(1, n + 1):
    for j in range(i, m + 1):
        cnt[i + j] += 1  # i+j 는 눈의 합
for i in range(n + m + 1):
    if cnt[i] > max:
        max = cnt[i]
for i in range(n + m + 1):
    if cnt[i] == max:
        print()
```

- 딕셔너리를 이용한 방법

```python
n = 4
m = 6

dic = {}
for i in range(1, n + 1):
    for j in range(1, m + 1):
        tmp = i + j
        if tmp not in dic.keys():
            dic[tmp] = 0
        dic[tmp] += 1
max_val = max(dic.values())

for k, val in dic.items():
    if val == max_val:
        print(k, end="")
```

## 3. 짚고 넘어갈 것

- **enumerate, items**

  1. list 에는 emumerate 라는 키 벨류를 모두 출력해주는 함수가 있다.

  ```python
  list = [1, 2, 3, 4, 5]
  for key, val in enumerate(list):
      print(key, val)

  # 0, 1
  # 1, 2
  # 2, 3
  # 3, 4
  # 4, 5
  ```

  2. 이와 비슷하게 Dictionary 에도 키와 벨류를 모두 출력해주는 items 함수가 있다.

  ```python
  dic = {'가':1, '나':2, '다':3,'라':4,'마':5}
  for key, val in dic.items():
      print(key, val)
  ```
